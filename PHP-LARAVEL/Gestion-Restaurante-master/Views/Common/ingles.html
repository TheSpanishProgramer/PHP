Sobre la Aplicación

Esta aplicación es una aplicación creada para la gestión de un negocio hostelero. Sus principales funcionalidades son:

  • Gestión de productos y categorias de manera rapida y sencilla.
  • Gestión de usuarios
  • Aplicación para clientes para automatizar pedidos.
  • Visor y gestion de pedidos a Cocina
  • Centro de trabajo para administracion y venta en mostrador/caja.

About App

This App is an App created for hostelry business management. It's main features are:

  • Fast and simple products and categories management.
  • Users management.
